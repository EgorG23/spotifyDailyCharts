##  Описание

Проект реализует ETL-пайплайн для загрузки, обработки и хранения данных из чарта Spotify Global Daily Chart в PostgreSQL.

---

## Перед использованием

1. Скачать данные: https://charts.spotify.com/charts/view/regional-global-daily/latest (необходимо авторизация через Spotify)
2. Заполнить CSV-файлами папку data/initial
3. Установить зависимости requirements.txt
```bash
pip install -r requirements.txt
```
4. Создать БД в PostgreSQL (таблица создается скриптом!)
5. Настроить соединение с БД: создать в корневой папке проекта файл .env и указать переменные DB_HOST, DB_PORT, DB_NAME, DB_USER, DB_PASSWORD
6. Подключить БД к PowerBI

---

##  Архитектура

- Extract: чтение CSV
- Transform: очистка и нормализация данных
- Load: загрузка в PostgreSQL через COPY (с проверкой на наличие данных в датасете)

---

## Особенности

- имитация получения данных из стороннего источника
- логирование всех ETL этапов
- SQL-скрипты, которые можно использовать в дальнейшем для аналитики

##  Структура данных

### Основная таблица - spotify_daily_charts

### Views (so far
